# House Price Prediction Using Machine Learning

This project predicts house prices using machine learning techniques. The model is trained on housing data and uses important features such as area, number of bedrooms, bathrooms, location-related details, and other property features to estimate the price of a house.

## Project Overview

House price prediction is a regression-based machine learning problem. The aim of this project is to analyze housing data, preprocess it, train a machine learning model, and predict house prices based on given input features.

## Dataset

The dataset contains housing-related features such as:

- Area
- Bedrooms
- Bathrooms
- Stories
- Main road access
- Guest room
- Basement
- Hot water heating
- Air conditioning
- Parking
- Furnishing status
- Price

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model Used

This project uses supervised machine learning regression models for house price prediction.

Common models used:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Steps Involved

1. Importing required libraries
2. Loading the housing dataset
3. Checking missing values
4. Encoding categorical columns
5. Splitting data into training and testing sets
6. Scaling/processing features if required
7. Training the machine learning model
8. Predicting house prices
9. Evaluating model performance

## Model Evaluation

The model can be evaluated using regression metrics such as:

- Mean Absolute Error
- Mean Squared Error
- Root Mean Squared Error
- R2 Score

## Features

- Data preprocessing
- Categorical data encoding
- Train-test split
- Regression model training
- House price prediction
- Model performance evaluation

## Output

The trained model predicts the estimated price of a house based on the input property details.

## Conclusion

This project demonstrates how machine learning can be used to predict house prices. Regression models help estimate property prices by learning patterns from historical housing data.
